# JavaDrill

このリポジトリは、Java の基礎文法やアルゴリズム理解を目的とした学習用プロジェクトです。  
日々の学習内容を小さな単位で積み重ね、基礎力の強化を目的として作成しています。

## 目的
- Java の基本文法の理解（クラス、メソッド、継承、例外処理など）
- コレクションやストリーム API の扱いに慣れる
- 基本的なアルゴリズム・データ構造の理解
- 今後の業務システム開発・基盤保守に活かすための基礎固め

## 技術スタック
- 言語: Java
- ビルド / 実行: JDK（ローカル環境）
- IDE: IntelliJ IDEA / VSCode など（任意）

## 内容
- 文法演習コード  
- 小規模なアルゴリズム・ロジックの実装  
- オブジェクト指向設計の理解を目的としたサンプルコード  

## 背景
実務では VBA / GAS / kintone などを用いた業務システム改善に携わっており、  
今後の開発案件や基盤システム改修にも活かせるよう、Java の基礎力向上を目的に継続学習しています。
